Introduction to CSS Media Types

CSS Media Types allow developers to apply styles depending on the type of device or media being used. The most common media types include:

  • all: Applies to all devices.
  • screen: Specifically for screens such as monitors, tablets, and phones.
  • print: Styles for printed documents.
  • speech: Styles for screen readers or voice-based devices.
