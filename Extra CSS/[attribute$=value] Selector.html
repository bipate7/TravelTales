CSS Attribute Selector: [attribute$=value]

This paragraph has an href ending with ".txt".

This paragraph has an href ending with ".pdf".

Explanation

The [attribute$=value] selector matches elements whose attribute value ends with a specific string.

  • The first paragraph's href ends with ".txt", so it does not match.
  • The second paragraph's href ends with ".pdf", so it gets a pink background and bold text.
  • This selector is commonly used for targeting file types in links, like PDFs or images.
