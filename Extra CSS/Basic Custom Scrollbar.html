Explanation

In this example, a custom scrollbar is applied using the following properties:

  • ::-webkit-scrollbar: Targets the scrollbar.
  • ::-webkit-scrollbar-thumb: Styles the scroll handle.
  • ::-webkit-scrollbar-track: Styles the scrollbar background.
