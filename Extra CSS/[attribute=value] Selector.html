CSS Attribute Selector: [attribute=value]

This paragraph has data-role="user" and does not match the selector.

This paragraph has data-role="admin" and matches the selector.

Explanation

The [attribute=value] selector targets elements with a specific attribute and value.

  • The first paragraph does not match because its data-role value is "user".
  • The second paragraph matches because its data-role value is "admin".
  • This is useful for creating precise styles for elements with specific roles or states.
