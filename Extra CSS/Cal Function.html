CSS calc() Function Example

Explanation of calc() Usage

  • Dynamic Width: The card width is calculated using the CSS calc() function.
  • Formula: calc(50% - 20px). This means the card takes 50% of the container's width and subtracts 20 pixels to add spacing.
  • Responsive Design: This ensures the card adapts to different screen sizes while maintaining proper spacing.
  • Max Width: We also add max-width: 400px to prevent the card from becoming too wide on larger screens.
