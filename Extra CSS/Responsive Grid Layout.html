Explanation

This layout is responsive, achieved using grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));.

  • auto-fit adjusts the number of columns based on available space.
  • minmax(200px, 1fr) ensures grid items are at least 200px wide but can expand to fill the container.
