CSS Attribute Selector: [attribute*=value]

This paragraph's title contains "Tutorial".

This paragraph's title contains "Guide" and matches the selector.

Explanation

The [attribute*=value] selector matches elements whose attribute value contains a specific substring.

  • The first paragraph's title contains "Tutorial", so it does not match.
  • The second paragraph's title contains "Guide", so it gets a blue background and italic text.
  • This selector is useful for targeting elements with partial matches in their attributes.
