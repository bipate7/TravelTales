Explanation of the Code

1. Grid Container:

  • display: grid;: This activates the CSS Grid layout for the container, making it a grid container.
  • grid-template-columns: repeat(3, 1fr);: This creates 3 equal-width columns. The 1fr is a fractional unit that divides the available space equally among the columns. The repeat(3, 1fr) syntax specifies that there will be 3 equal columns.
  • gap: 10px;: Adds a 10px gap between each grid item both horizontally and vertically.
  • padding: 20px;: Adds 20px padding inside the grid container, creating space between the containerâ€™s border and its items.

2. Grid Item:

  • background-color: lightblue;: Sets the background color of each grid item to light blue.
  • padding: 20px;: Adds 20px of padding inside each grid item, creating space around the text or content inside the grid items.
  • text-align: center;: Centers the content (the text) inside each grid item horizontally.
  • border-radius: 8px;: Rounds the corners of each grid item by 8px for a smoother, more polished look.
  • box-shadow: 0 4px 8px rgba(0, 0, 0, 0.2);: Adds a subtle shadow effect around each grid item. This shadow is offset vertically by 4px and spread by 8px, with a 20% opacity for a soft effect.

3. Heading (h1):

  • text-align: center;: Centers the heading text horizontally on the page.
  • color: #333;: Sets the color of the heading text to a dark gray (#333), which is easy to read and contrasts well with the background.

Result:

The page will display a 3-column grid with 6 items arranged within the grid. Each item will have a light blue background, centered text, rounded corners, and a subtle shadow effect. The grid will have gaps between items, and padding will be applied inside the grid container to add space around the grid items.
