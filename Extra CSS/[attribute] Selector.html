CSS Attribute Selector: [attribute]

This paragraph does not have a data-note attribute.

This paragraph has the data-note attribute.

Explanation

The [attribute] selector applies styles to elements that have a specific attribute, regardless of its value. Here:

  • The first paragraph has no data-note attribute, so no special styles are applied.
  • The second paragraph has the data-note attribute, so it gets a green background and bold white text.
