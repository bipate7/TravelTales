Z-Index CSS Example and Explanation

Explanation of the Code:

1. Container (.container):

  • position: relative; - This positions the container relative to its normal position, making it the reference point for any absolutely positioned child elements inside it.
  • width: 300px; and height: 200px; - These set the size of the container.
  • background-color: #f0f0f0; - Sets a light gray background color for the container, making it visually distinct.
  • border: 1px solid #ccc; - Adds a border around the container to visually separate it from other elements.

2. Box 1 (.box1):

  • position: absolute; - This positions the box relative to the nearest positioned ancestor, in this case, the .container.
  • z-index: 1; - The z-index controls the stacking order. Box 1 has a lower z-index than Box 2, so it appears behind it.
  • top: 50px; and left: 50px; - Positions Box 1 50px from the top and left within the container.
  • width: 100px; and height: 100px; - Sets the dimensions of Box 1.
  • background-color: red; - Gives Box 1 a red color.

3. Box 2 (.box2):

  • position: absolute; - Positions Box 2 absolutely within the container.
  • z-index: 2; - Box 2 has a higher z-index than Box 1, so it appears on top of Box 1.
  • top: 30px; and left: 80px; - Positions Box 2 slightly above and to the right of Box 1, so they overlap.
  • width: 100px; and height: 100px; - Sets the dimensions of Box 2.
  • background-color: blue; - Gives Box 2 a blue color.

4. Heading (<h1>):

  • text-align: center; - Centers the heading text horizontally.
  • color: #333; - Sets a dark gray color for the heading text for better readability.

5. Paragraph (<p>):

  • text-align: center; - Centers the explanatory text.
  • color: #555; - Sets the text color to a lighter gray for a softer look.

Summary of z-index Usage:

  • The z-index property determines the stacking order of elements. The element with the higher z-index will appear on top of elements with a lower z-index.
  • In this example, Box 1 has a z-index of 1 and appears behind Box 2, which has a z-index of 2.
  • This is why the blue Box 2 overlaps and appears on top of the red Box 1, even though they are in the same container.
