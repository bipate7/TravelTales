Explanation of the Code

1. Initializing the Counter:

  • counter-reset: item-counter;: This resets the counter to 1 at the beginning of the ordered list. It's like setting a starting point for the numbering.
  • list-style: none;: This removes the default numbers or bullets that browsers apply to ordered lists. We will be using a custom counter for this.
  • padding-left: 20px;: Adds a little padding to the left side of the list, creating space between the content and the left edge of the page.

2. Incrementing the Counter:

  • counter-increment: item-counter;: This property increments the counter by 1 each time a new <li> element is added. So, the first <li> gets the counter 1, the second one gets 2, and so on.

3. Displaying the Counter:

  • content: counter(item-counter) ". "; : The ::before pseudo-element is used to insert the counter value before the content of each <li>. It will display the counter number followed by a dot (e.g., "1. First item").
  • font-weight: bold;: This makes the counter number bold so it stands out more before the list item text.

Result:

The ordered list will display the numbers before each item, with the counter incrementing automatically, and the numbers will appear bold.
